# Baccarat Game (百家乐网页版)

这是一个基于 Web 的百家乐游戏项目，旨在提供简洁、现代的移动端游戏体验，包含完整的游戏逻辑和统计功能。

## 📅 最近更新 (Updates)

### 1. 🎨 界面更新 (UI Updates)
- **顶部导航优化**：新增了 **设置 (Settings)** 和 **信息 (Info)** 图标按钮，布局更加合理。
- **视觉增强**：增大了 **余额 (Balance)** 和 **统计数据 (Stats)** 的字体显示，关键信息一目了然。
- **控件规范**：统一了按钮高度，将输入框调整为 **行内显示 (Inline input)**，整体风格更加协调。
- **状态反馈**：优化了按钮点击态和禁用态的视觉反馈；**飞牌**按钮现采用**浅绿色**样式，与操作区分更明显。

### 2. 🧠 游戏逻辑 (Game Logic)
- **洗牌机制**：每靴牌最大局数在 **58-72** 局之间随机生成，达到上限自动洗牌。
- **最后一局提示**：在倒数第二局结束后，系统会弹出 **"最后一局"** 提示，允许玩家进行最后一次操作。
- **操作冷却**：飞牌功能增加了 **1.5秒** 的冷却时间，防止误触和快速点击。
- **投注互斥**：实现了 **庄家 (Banker)** 与 **闲家 (Player)** 投注互斥，防止同时下注。
- **即时扣款**：下注时立即扣除对应余额，逻辑更加严谨。
- **路单优化**：简化了结果历史记录逻辑，**最新结果显示在最左侧**，符合移动端阅读习惯。

### 3. ✨ 新功能 (Features)
- **飞牌 (Fly Card)**：允许玩家跳过当前局（不经下注直接开牌），方便观察路单。
- **取消下注 (Cancel Bet)**：支持在开牌前撤销当前下注，退回本金。
- **设置模态框**：点击设置图标可唤起弹窗，支持选择不同的 **限红 (Table Limits)**。
- **信息模态框**：点击信息图标可查看游戏说明及版本信息。

### 4. 👁️ 显示优化 (Display)
- **极简按钮**：下注按钮仅在 **激活/已下注** 状态下显示金额，未下注时保持清爽。
- **信息精简**：移除了冗余的开牌结果文字描述，界面聚焦于路单和走势。
- **并排布局**：本金显示与统计数据调整为同一行并排显示，提高屏幕空间利用率。

## 🚀 如何运行 (How to Run)

本项目为纯静态网页，无需复杂的构建环境。

1. **直接运行**：在浏览器中直接打开 `baccarat.html` 文件即可开始游戏。
2. **兼容性**：推荐使用 Chrome, Safari 或 Edge 等现代浏览器，并使用移动端模式 (Mobile View) 获得最佳体验。

## 🛠️ 技术栈 (Tech Stack)
- HTML5 / CSS3 / Vanilla JavaScript
- 无需外部依赖 (Zero Dependencies)
